## 프로젝트 실행 스크립트
```
$ cd /AngelHack-Trusty
$ npm install

$ cd /AngelHack-Trusty/backend
$ npm install

$ cd /AngelHack-Trusty/frontend
$ npm install

$ cd /AngelHack-Trusty
$ npm start
```
